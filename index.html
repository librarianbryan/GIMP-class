Layers is the most important feature.

The default visible dockable dialogue is Layers. Layers is the key feature that seperates an advanced image editor from a basic one. Open up the file, called BigIdeas. - Create a new layer - Move a layer UP or DOWN - Create a duplicate layer - Delete a layer - Anchor a layer - linkages - create a layer group

DO STUFF

Resizing Tools

Canvas vs. Image vs. Layer

Crop Tool [Shift+C]

This chops things.

Scale Tool [Shift+T]

This makes things larger or smaller.

Zoom Tool [Z]

This zooms in and out.

Move Tool [M]

This moves objects around.This is also your default mouse pointer.
1) Canvas is the size of the editable surface 2) Image is all the layers. 3) Layer is just one layer.

Selection Tools

Rectangle Tool [R]

This selects rectangles and squares.

Ellipse Tool [E]

This selects circles, fool.

Free Select Tool [F]

This is for free hand selections.

Paths Tool [B]

Creates modular selections that persist between layers.

Paths Tool vs. Scissor Select Tool

Color Tools

Colors area and Colors menu

Fuzzy Select Tool [U]

This selects continuous regions of color.

Select by Color Tool [Shift+O]

This all of one color in the layer.

Color Picker Tool [O]

This picks color from current layer.
